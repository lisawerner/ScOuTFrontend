Hilfe: Lexikon (Begriffe) Task Ein Task ist eine Aufgabe, Planungsschritt, ToDo, welche du erledigen musst um ein Event zu planen.

Beispiel: "Einkauf erledigen"

Unerledigte Tasks werden eingteilt in: "dringend" (Reminder ist aktiv) und "offen" (Reminder ist noch inaktiv)

 Reminder Ein Reminder ist eine Erinnerung, ab wann der Task dringend erledigt werden sollte. Du kannst den Reminder beim Erstellen des Tasks relativ zum Eventbeginn angeben. Event Ein Event ist eine Aktion, Veranstaltung oder sonstiger Termin, welches dein Team vorbereitet. Das Event kann mit Hilfe von Tasks geplant und vorbereitet werden. Template Aus Events können Templates (Vorlagen) erstellt werden. Um ähnliche Events zu erstellen, kann man diese aus dem Template generieren und sich so die Arbeit sparen alle Tasks wieder einzeln anzulegen. Team Ein Team ist eine Gruppe, ein Verein oder ein anderer Zusammenschluss von Personen, welches gemeinsam Events plant. Tutorial Sitemap Begriffe
